> **대상:** 시스템 관리자, 기관 담당자, 멘토, 예비창업자(지원자), 기존 창업자

## 1. 시나리오 개요

### 1.1 배경 설정

**"2026 모두의 창업"** 프로그램 운영을 통해 CASA 시스템의 전체 기능을 시연합니다.
실제 DB에 등록된 기관, 멘토, 지원자 데이터를 기반으로 합니다.

### 1.2 전체 흐름 요약

```
Phase A: 사전 설정 (관리자)
  → 현황 확인 → 프롬프트 관리 → 크레딧 충전

Phase B: 매핑 및 매칭 (관리자 + 기관 담당자)
  → 신규 프로젝트-기관 매핑 → 멘토 매칭 → 안내 메시지

Phase C: 예비창업 트랙 (지원자 + 멘토)
  → 아이디어 → AI 확장/보강/유사기업 → 캔버스 PDF
  → 3관점 평가 → 이의 제기 → 문서 7종 생성 → PPT 이미지
  → 멘토 피드백 → AI 재생성 → 배포 → 쇼케이스

Phase D: 창업 트랙 (지원자)
  → 사업 검토 → 경영 진단(SWOT) → 전략 수립 → 종합 보고서

Phase E: 멘토 작업
  → 워크스테이션 피드백 → 보고서 작성 → AI 요약 → 수당 확인 → 프로필 관리

Phase F: 검수 및 행정 (기관 담당자)
  → 보고서 확인/반려 → 수당 승인(개별/일괄/CSV) → 현황 확인

Phase G: 지원자 후속
  → 피드백 열람 → 좋아요 → 만족도 평가 → 기관 지원 신청

Phase H: 부가 기능
  → 다국어 → 다크모드 → 모바일 → 알림 → 멘토 개인 프로젝트
```

## 3. 시연 시나리오 (단계별)

### Phase A: 사전 설정 (관리자)

> **로그인:** admin@cbnu.ac.kr / admin1234!

#### A-1. 전국 현황 대시보드

1. `/ko/admin/overview` 접속
2. 확인: 활성 프로그램(2개), 등록 기관(4개), 승인 멘토(6명), 전체 프로젝트, 승인 대기 건수

**시연 포인트:** 시스템 전체 현황을 한눈에 파악

#### A-2. 프로그램 관리

1. `/ko/admin/programs` 접속
2. 기존 프로그램 2개 확인: "2026 모두의 창업", "2026 예비창업 멘토링"
3. 프로그램 상세 확인 (기간, 상태, 연결된 매핑 수)

**시연 포인트:** 연도/차수별 프로그램 단위 관리

#### A-3. 기관 및 사용자 관리

1. `/ko/admin/institutions` → 4개 기관 현황 확인 (지역, 유형, 승인 상태)
2. `/ko/admin/users` → 역할별 필터링 (전체/user/mentor/institution/대기중)
3. `/ko/admin/mentors` → 6명 멘토 목록, 전문분야, 승인 상태 확인

**시연 포인트:** 역할별 사용자 관리, 승인 워크플로우

#### A-4. 프롬프트 관리 ★

1. `/ko/admin/prompts` 접속
2. 카테고리별 프롬프트 목록: ideation / evaluation / document / startup / mentoring
3. `idea_expansion` 프롬프트 클릭 → `/ko/admin/prompts/[id]`
4. 시스템 프롬프트, 사용자 프롬프트 템플릿, 모델, temperature 확인
5. **"테스트"** 기능: 변수 입력 후 AI 응답 미리보기
6. **버전 히스토리**: 이전 버전 비교, 필요 시 **롤백**
7. **"캐시 동기화"** → Redis 캐시 갱신

**시연 포인트:** DB 기반 프롬프트 관리, 버전/테스트/롤백/캐시 동기화

#### A-5. 크레딧 관리

1. `/ko/admin/credits` 접속
2. 곽은서(kwak@cbnu.ac.kr) 선택 → 50 크레딧 충전
3. 김연광(kyk9248@cbnu.ac.kr) 선택 → 30 크레딧 충전
4. 사유: `2026 모두의 창업 참가자 초기 크레딧`

**시연 포인트:** 관리자가 개별 사용자에게 크레딧 할당

### Phase B: 매핑 및 매칭

#### B-1. 신규 프로젝트-기관 매핑 (관리자)

> **로그인:** admin@cbnu.ac.kr

1. `/ko/admin/mappings` 접속 → 기존 매핑 6건 확인
2. **"매핑 추가"** 클릭
3. 프로젝트 검색: `Connect Care` 입력 → **결과에서 선택**
4. 기관: `충북창조경제혁신센터`
5. 프로그램: `2026 모두의 창업`
6. 저장
7. 추가로 `예비창업자의 모험` (김연광) → 충남창조경제혁신센터 매핑

**시연 포인트:** 프로젝트 이름 검색(UUID 불필요), 다른 기관/프로그램으로도 매핑 가능

#### B-2. 멘토-프로젝트 매칭 (기관 담당자)

> **로그인:** cb@creativekorea.or.kr (충북센터)

1. `/ko/institution/dashboard` 접속 → 관할 프로젝트 현황
2. `/ko/institution/mentors` → 소속 멘토 3명 확인 (김멘토, 박멘토, 이멘토03)
3. `/ko/institution/matches` 접속 → 기존 매칭 현황
4. 신규 매칭: `Connect Care` ← `김멘토` (주멘토, 인공지능 전문)
5. 저장

**시연 포인트:** 기관이 직접 멘토-프로젝트 매칭, 멘토 전문분야 참고

#### B-3. 기관 일괄 메시지 발송 ★

1. `/ko/institution/messages` 접속
2. **"일괄 발송"** 탭
3. 대상: `전체 멘토`
4. 제목: `2026 모두의 창업 멘토링 안내`
5. 내용:
   ```
   안녕하세요, 충북창조경제혁신센터입니다.
   2026 모두의 창업 프로그램 멘토링이 시작되었습니다.
   멘토링 프로젝트 메뉴에서 배정된 프로젝트를 확인해주세요.
   3월 15일까지 첫 번째 피드백을 부탁드립니다.
   ```
6. **발송** → 3명 멘토에게 동시 발송

**시연 포인트:** 소속 멘토/지원자에게 일괄 메시지

#### B-4. 개별 메시지 발송 ★

1. **"새 메시지"** 클릭
2. 수신자: `곽은서` 검색 → 선택
3. 제목, 내용 작성 후 **발송**

**시연 포인트:** 이름/이메일 검색으로 수신자 선택, 쓰레드 답장

### Phase C: 예비창업(Pre-Startup) 트랙

> **로그인:** kwak@cbnu.ac.kr (곽은서) — 또는 신규 프로젝트 생성용 계정

#### C-1. 프로젝트 생성

1. `/ko/projects/new` 접속
2. 프로젝트명: `AI 기반 만성질환 코칭 서비스 헬스브릿지`
3. 트랙: **예비창업(Pre-Startup)**
4. 생성 → 프로젝트 상세 페이지

#### C-2. Stage 1 — 아이디어 (Idea) ★

##### AI 확장 (Expand)
1. **"아이디어"** 탭 → 원문 입력:
   ```
   AI 기반 만성질환 관리 코칭 서비스. 당뇨/고혈압 환자 1,800만 명의 자가관리를
   AI 코칭 엔진으로 지원. 환자용 앱(AI 코칭, 복약 알림, 식단 분석),
   의사용 대시보드(모니터링, EMR 연동). B2B SaaS + B2B2C 수익모델.
   국내 디지털 헬스 시장 15조원.
   ```
2. **"AI 확장"** 클릭 → **SSE 스트리밍**으로 실시간 생성
3. 구조화 결과: 문제/솔루션/타겟/차별점/채널/수익/비용/지표

**시연 포인트:** 비정형 텍스트 → BM 캔버스 자동 구조화 (실시간 스트리밍)

##### AI 보강 (Enhance)
1. 특정 섹션이 부족하면 **"AI 보강"** 클릭
2. 기존 내용을 더 풍부하게 개선

**시연 포인트:** Expand(처음 생성)와 Enhance(기존 보강) 차이

##### 유사 기업 분석
1. **"유사 기업 분석"** 클릭
2. AI가 국내외 경쟁/유사 서비스 분석 제시

**시연 포인트:** 경쟁 환경 자동 분석

##### 린 캔버스 PDF 내보내기
1. **"캔버스 PDF"** 클릭 → PDF 다운로드

**시연 포인트:** 시각적 산출물 즉시 생성

##### 확정 → Gate 1 통과

#### C-3. Stage 2 — 평가 (Evaluation) ★

##### 3관점 AI 평가
1. **"평가"** 탭 → **"평가 시작"**
2. 3관점 실시간 평가 (SSE 스트리밍):
   - **투자자**: 투자 매력도, 팀 역량, 시장 타이밍
   - **시장**: TAM/SAM/SOM, 경쟁 분석
   - **기술**: 실현 가능성, 확장성, 혁신성
3. 관점별 점수(0~100) + 종합 점수

**시연 포인트:** 3명의 AI 전문가 독립 평가

##### 평가 이의 제기 (Dispute)
1. **"이의 제기"** 클릭
2. 내용: `시장 관점에서 디지털 헬스 시장 성장률이 과소평가됨. 연평균 25% 성장 중`
3. AI 재평가

**시연 포인트:** 쌍방향 소통, 지원자 반론 기회

##### 확정 → Gate 2 통과

#### C-4. Stage 3 — 문서 생성 (Document) ★

7종 문서 순차 생성:

| # | 문서 | 설명 |
|---|------|------|
| 1 | 사업계획서 | 1,500~2,000단어 (SSE 스트리밍) |
| 2 | 피치 덱 | 슬라이드별 카피 |
| 3 | PPT 발표자료 | PowerPoint + **AI 이미지 생성** |
| 4 | 랜딩페이지 | Hero + Features + CTA |
| 5 | 리플릿 | 마케팅 전단지 |
| 6 | 인포그래픽 | 핵심 데이터 시각화 |
| 7 | K-Startup 신청서 | 정부 지원 양식 자동 채우기 |

**시연 순서:**
1. 각 문서 **"생성"** → SSE 스트리밍 실시간 확인
2. **미리보기** 확인
3. 사업계획서에 **"수정 요청"** → `재무 계획을 보수적 시나리오로 수정` → AI 재생성
4. PPT에서 **"이미지 생성"** → AI가 슬라이드별 이미지 생성
5. 각 문서 **"확정"**

**시연 포인트:** 아이디어 하나로 7종 문서, 수정→재생성 루프, PPT AI 이미지

##### 전체 확정 → Gate 3 통과

#### C-5. Stage 4 — 배포 (Deploy)

1. **"배포"** 탭 → **"랜딩페이지 배포"**
2. 공개 URL 생성 → Gate 4 통과

#### C-6. 쇼케이스 확인

2. 프로젝트가 공개 목록에 노출 → 클릭 시 공개 프로필

**시연 포인트:** AI 생성물이 쇼케이스로 즉시 공개

> **로그인:** kyk9248@cbnu.ac.kr (김연광) — 기존 `창업자` 프로젝트 사용
> 또는 곽은서의 `Connect Care` (startup) 사용

#### D-1. Stage 1 — 사업 검토 (Review)

1. 창업 트랙 프로젝트 진입 → **"검토"** 탭
2. 사업 현황 입력: 매출, MAU, 주요 지표

**시연 포인트:** 기존 사업 데이터 기반 AI 분석

#### D-2. Stage 2 — 경영 진단 (Diagnosis)

1. **"진단"** 탭 → **"진단 시작"**
2. AI SWOT 분석 + 경영 건강도 점수(0~100)
3. 개선 권고사항 확인 → **확정**

**시연 포인트:** SWOT 자동화, 경영 건강도 정량 평가

#### D-3. Stage 3 — 전략 수립 (Strategy)

1. **"전략"** 탭 → **"전략 생성"**
2. 단기/중기/장기 전략 + KPI + 리스크 명시
3. **확정**

**시연 포인트:** 데이터 기반 전략 자동 수립

#### D-4. Stage 4 — 종합 보고서 (Report)

1. **"보고서"** 탭 → **"보고서 생성"**
2. Review + Diagnosis + Strategy 종합 보고서
3. **DOCX 다운로드** 가능
4. **확정** → 프로젝트 완료

**시연 포인트:** 전 단계 종합 AI 보고서, 문서 내보내기
