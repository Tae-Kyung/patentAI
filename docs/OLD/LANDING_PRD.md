# CASA 랜딩 페이지 PRD (Landing Page Product Requirements Document)

> **목표:** 예비 창업자와 창업팀이 CASA 플랫폼의 가치를 즉시 이해하고, 회원가입(전환)으로 이어지도록 설계된 프로모션 랜딩 페이지
> **문서 버전:** 1.0
> **작성일:** 2026-02-16

---

## 1. 프로젝트 개요

**프로젝트명**
- CASA 플랫폼 공식 랜딩(홍보) 페이지

**목적**
- CASA 플랫폼의 핵심 가치(AI 기반 창업 가속화)를 직관적으로 전달
- 회원가입 전환율 극대화
- 충북대학교 창업지원 생태계 내 브랜드 인지도 구축
- 기존 대시보드/서비스 기능과 완전히 분리된 독립 홍보 페이지

**타겟 사용자**
- 1차: 아이디어는 있으나 사업화 방법을 모르는 **예비 창업자** (대학생, 대학원생)
- 2차: 사업계획서/IR 자료 고도화가 필요한 **창업보육센터 입주 기업**
- 3차: 빠른 프로토타이핑이 필요한 **창업 동아리 팀**
- 4차: 효율적 팀 관리가 필요한 **창업지원단 멘토**

---

## 2. 핵심 목표 및 성공 지표

### 비즈니스 목표
1. 월 회원가입 **200건 이상** 달성
2. 회원가입 전환율 **5% 이상** 달성
3. 평균 체류 시간 **2분 이상** 유지

### 사용자 목표
1. 30초 내 CASA가 무엇인지 이해
2. 1분 내 핵심 기능 파악
3. "나도 써보고 싶다"는 동기 부여
4. 행동 유도 (회원가입)

### 핵심 성공 지표 (KPI)
- 회원가입 전환율: **5%** 목표
- 페이지 이탈률: **45% 이하**
- 평균 체류 시간: **2분 이상**
- CTA 클릭률: **12% 이상**
- 스크롤 완주율(마지막 섹션 도달): **40% 이상**

## 3. 페이지 구조

### 단일 페이지 구조 (Single Page Application)

1. **네비게이션 바** - 고정 헤더, 섹션 앵커 링크
2. **히어로 섹션** - 첫인상, 핵심 가치 전달
3. **문제 제기** - 창업자의 페인포인트 공감
4. **솔루션 개요** - CASA 워크플로우 시각화
5. **핵심 기능 소개** - 4대 핵심 기능 상세
6. **AI 기술 차별화** - 멀티 AI 오케스트레이션
7. **숫자로 보는 성과** - 신뢰 구축 지표
8. **Before/After 사례** - 실제 변화 스토리
9. **사용자 후기** - 사회적 증거
10. **이용 프로세스** - 4단계 간단 플로우
11. **지원 기관 및 파트너** - 권위 요소
12. **FAQ** - 의심 해소
13. **최종 CTA** - 행동 유도
14. **푸터** - 연락처 및 정보

## 4. 섹션별 상세 요구사항

### 4.1 네비게이션 바

**목적:** 브랜딩 + 빠른 탐색 + 전환 유도 (스크롤 시 상단 고정)

**레이아웃**
- 상단 고정 (sticky), 스크롤 시 배경 블러 효과
- 좌측: CASA 로고 + 브랜드명
- 중앙: 섹션 앵커 링크 (기능, 프로세스, FAQ)
- 우측: 로그인 버튼 (ghost) + 시작하기 버튼 (primary)
- 모바일: 햄버거 메뉴

**콘텐츠**
```
[CASA 로고]  기능 소개 | 이용 방법 | FAQ     [로그인] [무료로 시작하기]
```

**디자인 가이드**
- 높이: 64px
- 배경: 투명 → 스크롤 시 bg-background/80 backdrop-blur
- 로고: 그라디언트 텍스트 또는 아이콘 + 텍스트 조합

---

### 4.2 히어로 섹션

**목적:** 3초 내 핵심 가치 전달 및 강렬한 첫인상

**레이아웃**
- 전체 화면 높이 (100vh)
- 좌측: 텍스트 + CTA (60%)
- 우측: 대시보드 목업 또는 애니메이션 (40%)
- 모바일: 세로 스택

**콘텐츠**
```
뱃지: "충북대학교 창업지원단 공식 AI 플랫폼"

헤드라인: "아이디어 하나로,
          사업을 시작하세요"

서브헤드라인:
"CASA는 AI가 아이디어 검증부터 사업계획서, 랜딩페이지까지
 자동으로 만들어주는 창업 가속화 플랫폼입니다."

핵심 수치 (인라인 뱃지 3개):
⚡ 아이디어→사업계획서 30분    📊 AI 다면 평가    🚀 원클릭 배포

[주요 CTA 버튼] 무료로 시작하기 →
[보조 CTA 버튼] 데모 영상 보기 ▶

하단 신뢰 요소 (작은 텍스트):
"✓ 무료 사용  ✓ 신용카드 불필요  ✓ 3분 만에 첫 결과 확인"
```

**우측 비주얼**
- CASA 대시보드 스크린샷 또는 목업
- 또는: AI가 문서를 생성하는 과정을 보여주는 간단한 애니메이션
- 부유하는 UI 카드 요소들 (아이디어 카드, 평가 점수, 문서 미리보기)

**디자인 가이드**
- 뱃지: 연한 primary/10 배경, primary 텍스트, 라운드
- 헤드라인: 48px→56px, 볼드, 그라디언트 텍스트 (primary → accent)
- 서브헤드라인: 20px, text-muted-foreground
- 배경: 미묘한 그리드 패턴 또는 도트 패턴 (opacity 5%)
- CTA 버튼: primary 색상, 56px 높이, 호버 시 미세 확대 + 그림자

### 4.3 문제 제기 섹션

**목적:** 창업자의 실제 페인포인트에 공감하여 "내 이야기"라고 느끼게 만들기

**레이아웃**
- 섹션 제목 + 3단 카드 그리드
- 하단 강조 메시지

**콘텐츠**
```
섹션 제목: "창업, 이런 어려움 겪고 계시지 않나요?"

카드 1: 💡→📄 "아이디어는 있는데, 사업계획서를 어떻게 쓰죠?"
설명: "아이디어는 넘치는데 체계적으로 정리하고 문서화하는 것이
      막막하게 느껴집니다."

카드 2: 🔍→❓ "내 아이디어가 정말 사업성이 있는 건지 모르겠어요"
설명: "주변 반응만으로는 확신이 서지 않고, 전문적인 검증을
      받고 싶은데 비용과 시간이 부담됩니다."

카드 3: ⏰→😫 "창업 준비에 시간이 너무 오래 걸려요"
설명: "사업계획서, IR자료, 홈페이지까지... 혼자서 다 준비하려면
      몇 달이 걸립니다."

하단 강조 메시지 (박스 처리):
"창업팀이 사업계획서 작성에 쓰는 평균 시간: 3~4주
 투자 유치 IR 자료 준비: 2~3주
 랜딩페이지 제작: 1~2주

 → 총 2개월의 준비 기간, CASA와 함께라면 하루면 됩니다."
```

**디자인 가이드**
- 배경: bg-muted/30
- 카드: 화이트 배경, 연한 보더, 아이콘 크게 (48px)
- 하단 강조: primary/5 배경, primary 좌측 보더 4px
- 텍스트 톤: 공감적, 부드러운 어조

### 4.4 솔루션 개요 섹션

**목적:** CASA 워크플로우를 한눈에 시각화하여 "이렇게 쉽다"는 인상 전달

**레이아웃**
- 섹션 제목 + 설명
- 4단계 수평 플로우 다이어그램 (화살표로 연결)
- 모바일: 세로 타임라인

**콘텐츠**
```
섹션 제목: "AI가 창업의 모든 과정을 함께합니다"
섹션 설명: "아이디어 입력 하나로, 검증부터 배포까지 자동으로 완성됩니다"

플로우:
[1. 아이디어 입력] → [2. AI 다면 평가] → [3. 문서 자동 생성] → [4. 배포 및 공유]
 "자유롭게 쓰면      "투자·시장·기술     "사업계획서,         "원클릭으로
  AI가 구조화"        3관점 검증"         피치덱, 랜딩페이지"  세상에 공개"

각 단계 하단 태그:
1: "AI 확장 · 구조화 · 멘토 피드백"
2: "멀티 AI · 점수화 · 개선 제안"
3: "자동 생성 · 수정 요청 · PDF 다운로드"
4: "랜딩페이지 배포 · 공유 링크 · 멘토 승인"
```

**디자인 가이드**
- 배경: 화이트
- 플로우 카드: 라운드 카드, 상단에 단계 번호 뱃지 (primary)
- 화살표: 점선 또는 그라디언트 화살표
- 호버 시: 카드 살짝 상승 + 그림자 강화
- 애니메이션: 스크롤 트리거로 좌→우 순차 fade-in

### 4.6 AI 기술 차별화 섹션

**목적:** 기술적 우위를 시각적으로 보여주어 신뢰 + 흥미 유발

**레이아웃**
- 중앙 다이어그램 (오케스트레이션 시각화)

**콘텐츠**

### 4.8 Before/After 사례 섹션

**목적:** 실제 사용 시나리오로 변화를 체감시키기

섹션 제목: "CASA와 함께한 변화"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

사례 1: 컴퓨터공학과 3학년 김민수 팀 (캠퍼스 배달 플랫폼)

Before:
- 아이디어는 있었지만 사업계획서 작성법을 몰라 3주째 정체
- 창업 경진대회 마감 임박, 제출 자료 미완성
- 팀원 4명이 각자 만든 문서가 일관성 없음

CASA 활용:
→ 아이디어 입력 후 20분 만에 구조화된 아이디어 카드 완성
→ AI 다면 평가로 미처 생각 못한 경쟁사 리스크 발견
→ 사업계획서 + 피치덱 자동 생성

After:
- 하루 만에 경진대회 제출 자료 완성
- AI 평가 피드백으로 수익 모델 보완 → 경진대회 우수상 수상
- 생성된 랜딩페이지로 초기 사용자 유치 시작

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

사례 2: 창업보육센터 입주 기업 (AI 헬스케어 스타트업)

Before:
- 기존 사업계획서가 있지만 정부지원사업 양식에 안 맞음
- IR 자료 준비에 외부 컨설팅 200만원 견적
- 랜딩페이지 제작 외주 별도 비용

CASA 활용:
→ 기존 아이디어 입력 후 정부지원사업 양식에 맞는 사업계획서 생성
→ 투자심사역 AI 관점의 피드백으로 약점 보완
→ 랜딩페이지 자동 생성 및 배포

After:
- 컨설팅 비용 0원으로 IR 자료 완성
- 멘토 피드백 반영 후 정부지원사업 선정
- 배포된 랜딩페이지로 B2B 고객 문의 유입

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

사례 3: 창업 동아리 '이노베이터스' (스터디 매칭 서비스)

Before:
- 아이디어 회의만 2달째 진행 중
- "이게 될까?"라는 확신 부족으로 실행 못함
- 기술적으로 뭘 만들어야 할지 감이 안 잡힘

CASA 활용:
→ 5가지 아이디어를 각각 AI 평가 → 가장 높은 점수의 아이디어 선택
→ 기술전문가 AI의 구현 로드맵 참고하여 개발 방향 설정
→ 자동 생성된 랜딩페이지로 사전 수요 조사 진행

After:
- 객관적 데이터 기반으로 아이디어 선정 → 팀 내 합의 도출
- 2주 만에 MVP 방향 설정 완료
- 랜딩페이지로 200명 사전 등록 확보 → 교내 창업 지원금 획득
```

**디자인 가이드**
- 배경: 화이트
- Before: 연한 레드/오렌지 톤 배경 (destructive/5)
- After: 연한 그린 톤 배경 (success/5)

### 4.12 FAQ 섹션

**목적:** 잠재적 의심과 우려 사항 해소

**레이아웃**
- 아코디언 형식 (8~10개 질문)
- 2단 컬럼 (데스크톱)

**콘텐츠**
```
섹션 제목: "자주 묻는 질문"

Q1. CASA는 무료인가요?
A. 네, 현재 충북대학교 소속 사용자에게 모든 기능을 무료로 제공합니다.
   AI API 비용은 플랫폼에서 지원합니다.

Q2. 어떤 종류의 아이디어든 가능한가요?
A. 네, IT 서비스, 제조업, 서비스업 등 분야에 관계없이 모든 창업 아이디어를
   입력할 수 있습니다. AI가 해당 분야에 맞춰 분석합니다.

Q3. AI가 생성한 문서의 품질은 어떤가요?
A. 사업계획서는 정부지원사업 양식을 준수하며, 실제 제출 가능한 수준입니다.
   또한 각 단계에서 사용자가 직접 검토·수정할 수 있어 원하는 품질을
   달성할 수 있습니다.

Q4. AI가 모든 것을 자동으로 결정하나요?
A. 아닙니다. CASA는 'Human-in-the-Loop' 방식으로, AI는 초안을 만들고
   모든 중요 결정은 사용자가 검토·승인합니다. 4개의 승인 게이트를 통해
   사용자가 전체 프로세스를 제어합니다.

Q5. 생성된 문서를 수정할 수 있나요?
A. 네, 모든 문서는 섹션별로 수정 요청이 가능하고, 전체 재생성도 할 수 있습니다.
   원하는 결과가 나올 때까지 반복 수정이 가능합니다.

Q6. 멘토 피드백은 어떻게 받나요?
A. 각 단계에서 '멘토 피드백 요청' 버튼을 클릭하면 담당 멘토에게 알림이 갑니다.
   멘토는 대시보드에서 피드백을 작성하고, 사용자는 이를 확인·반영할 수 있습니다.

Q7. 랜딩페이지는 실제로 인터넷에 공개되나요?
A. 네, 배포 승인 후 실제 URL이 생성되어 누구나 접속할 수 있습니다.
   Vercel 인프라를 통해 안정적으로 호스팅됩니다.

Q8. 내 아이디어의 보안은 안전한가요?
A. 모든 데이터는 암호화되어 저장되며, Row Level Security를 통해
   본인과 허가된 멘토만 접근할 수 있습니다.
   아이디어가 외부에 노출되거나 학습 데이터로 사용되지 않습니다.

Q9. 영어로도 사용 가능한가요?
A. 네, 한국어와 영어를 지원합니다. 언어 설정에서 전환 가능하며,
   AI 생성 문서도 선택한 언어로 작성됩니다.

Q10. 충북대학교 소속이 아니어도 사용할 수 있나요?
A. 현재 MVP 단계에서는 충북대학교 소속 사용자를 대상으로 운영 중이며,
   향후 다른 대학 및 일반 사용자로 확대할 예정입니다.
```

**디자인 가이드**
- 배경: 화이트
- 아코디언: 클릭 시 부드럽게 펼쳐짐 (300ms ease)
- 질문: 볼드, 16px
- 답변: 일반, 14~15px, muted-foreground

## 5. 전체 디자인 시스템

### 컬러 팔레트

기존 shadcn/ui 테마를 확장 활용합니다.

```
Primary: hsl(var(--primary))  — 메인 CTA, 강조 요소
Accent: 보조 강조색 (그린 또는 퍼플 계열)
Background: hsl(var(--background))
Foreground: hsl(var(--foreground))
Muted: hsl(var(--muted)) / hsl(var(--muted-foreground))
Destructive: hsl(var(--destructive)) — Before 상태

그라디언트:
- 히어로 텍스트: primary → accent
- CTA 섹션 배경: primary/5 → accent/5
- AI 섹션 배경: slate-900 → slate-800 (다크 섹션)
```

### 타이포그래피

```
히어로 헤드라인: 48px (모바일: 36px), 볼드, tracking-tight
섹션 제목 (H2): 36px (모바일: 28px), 볼드
소제목 (H3): 24px (모바일: 20px), 세미볼드
본문: 16px~18px, 일반, leading-relaxed
캡션/보조: 14px, muted-foreground
CTA 버튼 텍스트: 18px, 세미볼드

폰트: Pretendard (한글), Inter (영문) — 시스템 폰트 fallback
```

### 간격 시스템

```
섹션 간 간격: py-20 (80px) ~ py-24 (96px)
섹션 내 제목→콘텐츠: mb-12 ~ mb-16

- **전환**: 모든 애니메이션 300ms ease, prefers-reduced-motion 존중
- **과도한 애니메이션 지양**: 단순하고 의미 있는 모션만 사용

### CTA 배치 (총 5곳)

1. **네비게이션**: 항상 노출 — "무료로 시작하기"
2. **히어로**: 첫 화면 — "무료로 시작하기" + "데모 영상 보기"
3. **사용자 후기 하단**: 중간 전환 — "나도 시작하기"
4. **이용 프로세스 하단**: 행동 촉구 — "지금 무료로 시작하기"
5. **최종 CTA 섹션**: 마무리 전환 — "무료로 시작하기"

### 신뢰 구축 요소 배치 순서

1. 히어로: 핵심 수치 뱃지 (30분, AI 평가, 원클릭 배포)
2. 숫자 성과: 상세 지표
3. Before/After: 구체적 변화 스토리
4. 사용자 후기: 실제 목소리
5. 지원 기관: 공식 파트너

### 심리적 저항 제거 요소

1. "무료" 강조 — 모든 CTA에 "무료" 명시
2. "신용카드 불필요" — 금전 부담 제거
3. "Human-in-the-Loop" — AI에 대한 불안 제거 (내가 컨트롤)
4. "30분" — 시간 투자 부담 최소화
5. FAQ — 보안, 품질, 수정 가능 여부 안내
6. 보안 메시지 — 아이디어 유출 우려 해소

### 긴급성/동기부여 요소

- "아이디어가 있다면, 지금 바로 시작하세요"
- "30분 후, 당신의 아이디어가 사업계획서가 됩니다"
