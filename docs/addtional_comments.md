1) 외부도면 이미지 추가 - 해석
특허 명세서를 기반으로 도면을 작성하는것을 좋다. 
업무상 고객이 외부 상세 도면을 추가하는 경우가 많이 있다. 
도면생성이 아니라 기존 외부 도면을 추가하고, 해당 도면을 설명하거나 해당 도면을 간소화해서 재생성하는 기능이 있으면 좋을것 같다.  

2) 오류사항
- 도면 이미지 생성시 하단에 잘못된 Fig. 번호 내용이 추가됨
- KIPO 도상 이라는 문구나 라벨이 추가되는 오류가 가끔 발생함
- 최초 3개의 도면을 계획했는데, 더 많은 도면이 그려짐. 계획과 도면 생성 부분이 연결되어 있는지 확인이 필요
- 도면 재생성을 누르면 재생성 과정에 대해서 표시되지 않음
- 프로젝트 목록 : 새로운 프로젝트 추가시, 중복된 카드가 나타나는 현상 발생
